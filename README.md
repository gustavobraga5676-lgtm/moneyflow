# MoneyFlow

Sistema de controle financeiro desenvolvido em Python utilizando CustomTkinter, SQLite e Matplotlib.

## Funcionalidades

- Dashboard Financeiro
- Cadastro de Receitas
- Cadastro de Despesas
- Histórico de Transações
- Pesquisa de Movimentações
- Análise Financeira
- Relatórios em TXT
- Metas Financeiras
- Gráficos Financeiros

## Tecnologias Utilizadas

- Python
- CustomTkinter
- SQLite
- Matplotlib
- Tkinter

## Estrutura do Projeto

moneyflow.py
moneyflow.db
README.md

## Como Executar

1. Instale as dependências:

pip install customtkinter matplotlib

2. Execute o sistema:

python moneyflow.py
